# Boterkoek

## Recipe Information

- **Cook Time:** 25 min
- **Course:** Dessert
- **Difficulty:** Easy

## Ingredients

- 350 gram bloem
- 3 ons boter
- 2 ons suiker
- snufje zout
- geraspte citroenschil

## Instructions

1. Verwarm de oven voor op 160 graden.
2. Veel kneden!
3. In vorm doen
4. Beetje eierdooier erover
5. Bakken gedurende 25 minuten op 160 graden
